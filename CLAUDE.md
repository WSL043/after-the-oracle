# Claude Guidance

This repository contains the `after-the-oracle` Agent Skill at `skills/after-the-oracle/SKILL.md`.

When the user asks for planning, review, public release preparation, high-stakes AI work, weak-model verification, decision briefs, or multi-pass agent workflows, use that skill first.

If the skill is not installed in Claude Code, read `skills/after-the-oracle/SKILL.md` directly and follow its operating defaults:

- plan before execution,
- separate execution from verification,
- prepare judgment briefs for irreversible or asymmetric-risk choices,
- keep project state in files,
- verify current claims from primary sources.
