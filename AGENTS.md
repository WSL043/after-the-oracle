# Agent Instructions

Use `skills/after-the-oracle/SKILL.md` when work is public, irreversible, high-cost, multi-step, or likely to require verification across several AI turns.

Default operating rules:

- Separate planning from execution for medium or larger tasks.
- Keep decisions, assumptions, progress, and TODOs externalized in files.
- Use independent verification or adversarial review before treating important output as done.
- When the task touches legal, medical, financial, security, credential, deletion, publication, or other irreversible areas, prepare options and risks for human judgment.
- Do not invent current facts, tool behavior, prices, model names, or compatibility claims. Verify with primary sources when they could have changed.

For this repository, keep README language clear, concrete, and human-centered. This project is about preserving human judgment, not promising autonomous correctness.
