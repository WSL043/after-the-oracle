# Security

After the Oracle is instruction-only, but instructions can still shape tool behavior.

Please report issues if the Skill or adapter files could cause an agent to:

- expose secrets,
- perform irreversible actions without human review,
- bypass security checks,
- treat unverified model output as authoritative,
- encourage unsafe legal, medical, financial, credential, or security decisions.

Do not include real secrets, API keys, private logs, or personal data in issues or pull requests.
