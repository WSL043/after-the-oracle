## Why This Name

The previous name, `Judgment Harness`, was accurate but emotionally cold. It described the mechanism, not the wound.

This project begins from a more human situation: the trusted, expensive, high-performing model is unavailable, and the user still has to make decisions. The source Skill calls this "after the judgment layer is gone" and ends with a voyage image. `After the Oracle` connects both:

- the oracle is the model that once carried judgment,
- the departure is the loss of permanent access,
- the compass is the procedure that lets the human keep navigating.

This name gives the repository a story before it gives it a system.

## Rejected Names

- `Judgment Harness`: technically clear, but not memorable enough.
- `Complete Judgment System`: too heavy and doctrinal.
- `Human Compass`: warm, but less specific to AI and less dramatic.
- `Decision Harness`: already crowded in enterprise AI/governance language.
